WELCOME TO AIRBNC API

The RESTful API for property listings, users and reviews

Endpoints Overview

  • GET /api/properties
  • GET /api/properties/:id
  • GET /api/properties/:id/reviews
  • GET /api/users/:id
  • POST /api/properties/:id/reviews
  • DELETE /api/reviews/:id

GET /api/properties

Returns a list of all properties. Supports filtering by price and property type.

Query parameters

  • property_type (string)
  • minprice (number)
  • maxprice (number)

Example request

GET /api/properties?property_type=Apartment&minprice=50&maxprice=150

Example response

{
  "properties": [
    {
      "property_id": 1,
      "property_name": "Modern Apartment in City Centre",
      "location": "London, UK",
      "price_per_night": 120,
      "property_type": "Apartment",
      "host": "Alice Johnson"
    }
  ]
}

GET /api/properties/:id

Returns a single property by its ID.

Example request

GET /api/properties/1

Example response

{
  "property": {
    "property_id": 1,
    "property_name": "Modern Apartment in City Centre",
    "location": "London, UK",
    "price_per_night": 120,
    "description": "...",
    "host": "Alice Johnson",
    "images": ["https://..."]
  }
}
